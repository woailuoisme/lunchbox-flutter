3.1 您通过本服务下达的订单，即视为您向我们发出的购买要约。当您支付成功且收到我们的确认通知时，合同即告成立。

3.3 如因设备故障、缺货等原因导致无法为您提供商品，我们将尽快联系您并为您办理退款。

第二部分：隐私政策

本隐私政策将帮助您了解我们如何收集、使用、存储和保护您的个人信息。

1. 我们收集的信息

为了向您提供服务，我们需要收集以下信息：

  • 账号信息：为了完成注册和登录，我们需要收集您的手机号码。如您选择第三方登录（如微信、Google），我们将收集您的第三方账号标识（OpenID）、昵称及头像。
  • 位置信息：为了向您推荐附近的贩卖机设备并提供导航服务，我们需要获取您的精确地理位置信息。您可以在系统设置中随时关闭此权限，但这将影响相关功能的使用。
  • 订单与交易信息：为了处理您的订单、支付及售后服务，我们会收集您的订单详情、支付记录、收货信息等。
  • 设备信息：为了保障账号安全、进行统计分析及排查故障，我们会收集您的设备型号、操作系统版本、唯一设备标识符（IMEI/Android ID/IDFA等）、IP地址、网络状态等信息。
  • 日志信息：我们会自动收集您使用本服务的详细情况，作为网络日志保存，包括您的搜索查询内容、访问时间、浏览记录、点击记录等。
  • 剪贴板信息：在您参与口令码活动或分享链接时，我们需要访问您的剪贴板以读取相关口令或链接，以便为您提供便捷的服务。
  • 相机/相册权限：当您使用扫码取餐、上传头像或反馈问题上传图片时，我们需要获取您的相机或相册权限。

2. 信息的存储

2.1 我们将在中华人民共和国境内收集和产生的个人信息存储在境内。如需跨境传输，我们将严格遵守相关法律法规规定。

2.2 我们仅在为您提供服务所必需的期限内保留您的个人信息，法律法规另有规定的除外。

3. 信息的使用

我们将把收集的信息用于以下用途：

  • 向您提供各项业务功能和服务；
  • 身份验证、安全防范、诈骗监测、存档和备份用途；
  • 根据您的位置和偏好，向您推荐更符合您需求的商品或服务；
  • 邀请您参与有关我们产品和服务的调查；
  • 改善我们的产品和服务体验。

4. 第三方SDK接入说明

为保障应用相关功能的实现与应用安全稳定的运行，我们可能会接入由第三方提供的软件开发包（SDK）实现相关目的。我们会对合作方获取信息的软件工具开发包（SDK）进行严格的安全监测，以保护数据安全。

我们接入的第三方SDK主要包括：

  • 支付服务（Stripe / 微信支付）：用于处理订单支付，可能收集您的支付信息、设备信息。
  • 地图服务（Google Maps / 高德地图 / 腾讯地图）：用于提供定位和导航服务，可能收集您的设备信息、位置信息。
  • 统计分析（Firebase Analytics / 腾讯移动分析）：用于应用数据统计和分析，可能收集您的设备信息、应用使用记录。
  • 推送服务（Firebase Cloud Messaging / 阿里云推送）：用于向您发送订单状态、活动通知，可能收集您的设备信息。
  • 社交分享（微信SDK）：用于支持微信登录及分享功能，可能收集您的设备信息、网络信息。

5. 您的权利

根据相关法律法规，您拥有以下权利：

  • 访问和更正：您可以随时登录账号访问或修改您的个人资料。
  • 删除：在特定情形下，您可以向我们提出删除个人信息的请求。
  • 撤回同意：您可以随时在系统设置中更改授权范围或撤回您的授权。
  • 注销账号：您可以通过【设置】-【注销账号】申请注销您的账号。注销后，我们将停止为您提供服务并删除您的个人信息。

6. 未成年人保护

我们非常重视未成年人的个人信息保护。若您是未满14周岁的未成年人，应在监护人监护、指导下阅读本协议，并在征得监护人同意后使用本服务。

7. 协议的变更

我们可能会适时对本协议进行修订。当协议条款发生变更时，我们会在版本更新时向您提示，并再次征得您的同意。
